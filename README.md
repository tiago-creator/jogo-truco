# Jogo de Truco Online

Projeto mobile first para truco online no navegador.

## Stack

- Client: Vite + TypeScript + Phaser
- Server: Node.js + TypeScript + Socket.IO
- Shared: tipos e regras compartilhadas

## Rodando

```bash
npm install
npm run dev
```

Client: `http://localhost:5173`

Server: `http://localhost:3000`
